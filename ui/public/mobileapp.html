Sync SDK Integration

To start with an example Sync app, clone/fork the repo at https://github.com/feedhenry-templates/sync-cordova-app. If you already have an App, you can integrate Sync into it.

Start by downloading the sync-config.json file and placing it in !!SOME FOLDER!!. Or alternatively, copy and paste the below config into !!SOME FILE!!

{{syncconfig}}

Add the Sync SDK to the App. This example is using npm for dependencies

npm install --save fh-sync-js@latest

Load the SDK config file & initialise the Sync SDK. This example is using AngularJS

angular
  .constant('fhSync', require('fh-sync-js'))
  .constant('fhSyncConfig', require('../../sync-config.json'))

fhSync.init(fhSyncConfig);

For info on how to use the Sync SDK, see the example app at https://github.com/feedhenry-templates/sync-cordova-app or the Sync Documentation at http://feedhenry.org/docs/
